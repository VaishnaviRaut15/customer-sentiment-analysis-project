# Customer Sentiment Analysis Project

 # Project Overview
This project is a Machine Learning-based Customer Sentiment Analysis system developed using Python and Jupyter Notebook.

The main objective of this project is to analyze customer reviews or feedback and classify the sentiment as:

- Positive 😊
- Negative 😞
- Neutral 😐

This project helps businesses understand customer opinions and improve customer satisfaction using sentiment prediction techniques.

---

#  Features

- Data preprocessing and cleaning
- Text analysis using NLP techniques
- Sentiment classification
- Machine Learning model training
- Accuracy evaluation
- Data visualization

# How the Project Works

1. Load customer review dataset

3. Convert text into numerical features
4. Train machine learning model
5. Predict sentiment
6. Visualize results and accuracy
